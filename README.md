**Günce**, anılarınızı sadece saklayan değil, onları hatırlayan ve sizinle paylaşan modern bir dijital asistanıdır. **Gemini AI** destekli "Hafıza Sohbeti" sayesinde, geçmişteki duygularınızı sorgulayabilir ve kendi gelişim yolculuğunuzda bilge bir yoldaş edinebilirsiniz. 🌌📖

---

## 🌟 Öne Çıkan Özellikler

| ✨ Özellik | 📝 Açıklama |
| :--- | :--- |
| **Hafıza Sohbeti (AI Chat)** 🧠 | Anılarınızı hatırlayan Gemini ile sohbet edin. Geçmişinizi sorgulayın, dertleşin ve analizler alın. |
| **Zen Temaları (5 Mod)** 🎨 | Lavanta, Doğa, Gece, Gül ve Çöl temalarıyla tüm buton, ikon ve gölgelerin değiştiği yaşayan arayüz. |
| **Duygu Zinciri (Heatmap)** 📊 | GitHub stili 90 günlük interaktif ruh hali takvimi. Görselleştirilmiş duygusal yolculuk. |
| **AI Yazma Yönergeleri** ✍️ | Gemini AI ile her gün size özel yaratıcı yazma teknikleri ve hikaye ipuçları. |
| **AI Duygu Analizi** 🧠 | Metinlerden otomatik ruh hali belirleme ve kişiselleştirilmiş duygusal özetler. |
| **Görsel Hafıza** 📸 | Anılarınıza cihaza entegre **Kamera** veya **Galeri** çekimleriyle görsellik katın. |
| **Sesli Duygular** 🎙️ | Yazamadığınız anlarda sesinizi anılarınıza ekleyerek ruhunuzu yansıtın. |
| **Anı Haritası** 🗺️ | Konum tabanlı anı takibi ve otomatik adres çözümleme (Reverse Geocoding). |

<br>

## 🚀 Hızlı Kurulum

1. **Projeyi indirin:**
```bash
git clone https://github.com/mehmeteminyilmaz/gunce.git
cd gunce
```
2. **Gerekli paketleri çekin:**
```bash
flutter pub get
```
3. **Uygulamayı başlatın:**
```bash
flutter run
```

<br>

## 🛠️ Teknolojik Altyapı

> 🧠 **Zeka:** `Gemini 1.5/2.5 Flash API` (Hafıza Bilinci ile Korunmuş Sohbet).  
> ✨ **UI:** `Glassmorphism Design`, `Dynamic Palette Swap`, `Custom Page Transitions`.  
> 💿 **Veritabanı:** `Hive` (Hızlı, Güvenli, Lokal Depolama).  
> 🗺️ **Lokasyon:** `Geolocator` & `Google Maps API Proxy`.

<br>

## 🌱 Gelişim Yolculuğu (Planlananlar)

- 🎙️ **Sesli Metin (STT):** Ses kayıtlarınızın otomatik olarak yazıya dökülmesi ve AI ile özetlenmesi.
- 📖 **PDF Dışa Aktar:** Tüm anılarınızı şık bir dijital kitaba (PDF) dönüştürme.
- ☁️ **Bulut Senkronizasyonu:** Firebase altyapısı ile çoklu cihaz ve yedekleme desteği.
- 📊 **Duygusal Karşılaştırmalar:** Ay bazlı ruh hali değişim grafiklerinin karşılaştırmalı analizi.

*Siz de bu yolculuğun bir parçası olabilir, projeyi yıldızlayarak (⭐) destek olabilirsiniz!*
